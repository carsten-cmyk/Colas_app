Brug git-agenten til at committe alle ændringer siden sidste commit.

Git-agenten vil:
1. Vise hvad der er ændret (`git status` + `git diff --stat`)
2. Gruppere filer efter app og type
3. Foreslå commit-besked
4. Vente på din godkendelse inden commit køres

Husk: aldrig auto-push — kun commit lokalt.
